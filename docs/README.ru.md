# Документация Nrgy.js

## Назначение

Эта директория является основной продуктовой документацией Nrgy.js. Она
объясняет, как использовать библиотеку, как устроена её архитектурная модель и
как вносить изменения в репозиторий.

## Рекомендуемый порядок чтения

1. Начать с [Введения](./introduction.ru.md).
2. Затем перейти к [Быстрому старту](./quick-start.ru.md).
3. После этого пройти [Core](./core/README.ru.md) и
   [Архитектуру](./architecture/README.ru.md).
4. Затем перейти к [MVVM и Controllers](./mvvm/README.ru.md).
5. Во время внедрения использовать [Интеграции](./integrations/README.ru.md),
   [Рецепты](./recipes/README.ru.md), [Миграцию](./migration/README.ru.md) и
   [FAQ](./faq/README.ru.md).
