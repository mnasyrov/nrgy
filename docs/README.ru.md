# Документация Nrgy.js

## Назначение

Этот раздел является точкой входа в проектную документацию Nrgy.js.

## Общая информация

Документация репозитория разделена на две основные области:

- документация пакетов, описывающая опубликованные пакеты `@nrgyjs/*`;
- документация для разработчиков, описывающая правила разработки, стиль кода и
  стандарты документации.
