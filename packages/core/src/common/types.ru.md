# `types.ts`

## Назначение файла

Модуль содержит общие вспомогательные типы, используемые в разных частях
`@nrgyjs/core`.

## Общая информация

Хотя файл не содержит runtime-логики, он задает базовые контракты для
универсальных объектов, функций и пользовательских стратегий сравнения
значений. Эти типы участвуют в сигнатурах реактивного API и утилит.

## Концептуальная архитектура

Модуль представляет собой тонкий слой общих TypeScript-типов:

- `AnyObject` используется там, где нужен словарь без строгой структуры.
- `AnyFunction` применяется во внутренних местах, где требуется универсальный
  callable-тип.
- `ValueEqualityFn<T>` стандартизует сигнатуру функций сравнения значений.

Этот файл зависит только от TypeScript и не имеет внешних зависимостей.

## Описание публичного API

### `AnyObject`

- Тип `Record<string, any>`.
- Используется как универсальный словарь объектов.

### `AnyFunction`

- Тип `(...args: any[]) => any`.
- Используется как обобщенный тип функции.

### `ValueEqualityFn<T>`

- Сигнатура `(a: T, b: T) => boolean`.
- Определяет контракт функции сравнения двух значений одного типа.

## Примеры использования

```ts
import type { ValueEqualityFn } from '@nrgyjs/core';

const compareNumbers: ValueEqualityFn<number> = (a, b) => a === b;
```
